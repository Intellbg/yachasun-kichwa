# Gramática 2: Adjetivos

En el kichwa tambien se puede calificar sustantivos para identificar sus cualdidades. Para esto al igual que en el 
español se utilizan adjetivos. A continuación se presenta una lista de adjetivos comunes que podrian ser utlizados
para calificar.

<VocabularyTable lecture="animals" tag='adjective' />

Una oración completa tiene la siguiente estrucutra:
<table>
<tbody>
<tr>
<th>Sustantivo conjugado</th>
<th>Calificativo conjugado</th>
<th>Verbo</th>
</tr>
<tr>
<td><strong>Kuchi</strong>ka</td>
<td><strong>wira</strong>mi</td>
<td>kan</td>
</tr>
</tbody>
</table>
Esto significa El chancho es gordo

Un ejemplo adicional 
- Atukka millaymi kan | El lobo es bravo
- Ukuchaka uchillami kan | El ratón es pequeño
