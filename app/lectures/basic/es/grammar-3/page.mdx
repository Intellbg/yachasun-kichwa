## Género

A pesar de que en el Kichwa no existan artículos especificos como en el español para ser usados con los 
sustativos para denotar el género del sustantivo la distinción de género se realiza con las palabras
*warmi* para femenino y *kari* para masculino. Nota que estas palabras son justamente muujer y hombre respectivamente.

Para realizar la distinción se coloca la palabra antes del sustantivo como por ejemplo

 <ImageFlipCard

El prefijo warmi tambien sule ser referencia a fino y delicado calificando de esta manera al sustantivo. Mientras 
el prefijo kari suele usarse para referenciar algo rústico, grande, bursco. Un ejemplo.

*warmi*sisa -> Flor bonita y delicada

*kari*sisa -> flor rústica, grande
