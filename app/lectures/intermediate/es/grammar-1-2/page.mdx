# **Las partículas pak y nkapak (Shimikukuna)**
En esta lección aprenderás a usar dos partículas importantes del Kichwa: ***-pak*** y ***-nkapak***. Estas partículas se emplean para modificar sustantivos, adjetivos y especialmente verbos, permitiendo expresar pertenencia, posición, objetivos o razones de una acción. Su uso correcto es esencial para enriquecer tus habilidades en el idioma.
<hr />  
## ***-pak***
La partícula ***-pak*** se utiliza para indicar pertenencia o posición, y siempre se combina con sujetos personales para formar pronombres posesivos.

## ***-nkapak***
La partícula ***-nkapak*** se utiliza para indicar el objetivo o la razón de una acción. Siempre se combina con verbos, y es útil para explicar por qué o para qué se realiza una actividad.
