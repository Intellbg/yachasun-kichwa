# **Partículas de pregunta y respuesta (Tapuy shimikukuna kutichi shimikukunapash)**
En esta lección aprenderás las partículas más comunes para formular preguntas y responder en Kichwa. Estas partículas son esenciales para entablar conversaciones y comprender las respuestas en este idioma. En Kichwa, no se utilizan signos de interrogación, por lo que las partículas cumplen un papel fundamental para indicar preguntas y respuestas.
## ***-tak***
Se utiliza para hacer preguntas, especialmente con palabras interrogativas. Coloca la partícula ***-tak*** después de estas palabras para formular preguntas.

<hr />
## ***-ka***
También se utiliza para formular preguntas, pero de forma más sencilla y directa, sin necesidad de palabras interrogativas.

<hr />
## ***-chu***
Esta partícula se usa para preguntas, pero no con palabras interrogativas, sino con pronombres personales o sustantivos. Además:
