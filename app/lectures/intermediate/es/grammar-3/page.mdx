# **Las partículas tak y ta (Shimikukuna)**
En esta lección aprenderás a usar dos partículas esenciales en Kichwa: -tak y -ta. Estas partículas son fundamentales para formular preguntas y afirmar respuestas con énfasis. 
<ComicSpeechBubble text="En kichwa no utilizamos signo de pregunta, la partícula lo remplaza." character = "humuFeliz"/>
<hr />
# ***-tak***
La partícula ***-tak*** se utiliza para hacer preguntas. Se coloca al final del verbo o la palabra clave en la oración para reemplazar los signos de interrogación.
<hr />
# ***-ta***
La partícula ***-ta*** se utiliza para afirmar o dar énfasis a una respuesta. Generalmente acompaña al objeto directo en la oración.
